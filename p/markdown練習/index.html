標題 / 粗.斜體 / 序列

The following HTML <h1>—<h6> elements represent six levels of section headings. <h1> is the highest section level while <h6> is the lowest.

H1 一級標題#

H2 二級標題##

H3 三級標題###

H4 四級標題####

H5 五級標題#####
H6 六級標題######

粗體寫法 : 文字前後都加上**

斜體寫法 : 文字前後都加上*

😱Markdown語法不能文字改顏色，也不能粗體和斜體一起使用。

有序序列 :

  1. 這是1 - 輸入1.空格 再打內容
  2. 這是2 - 輸入2.空格,也可以輸入1.空格，MD會自己做運算。

無序序列 :

  • HELLO - 輸入 * 空格 再打內容
